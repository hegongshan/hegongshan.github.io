进阶:
如果多次调用这个函数，你将如何优化你的算法？

方法一：位运算+循环

n对2取模，相当于和1做按位与运算，即 n % 2 等价于 n & 1。

时间复杂度为O(1)：由于运算时间与n的位数有关，而int型数据只有32位，因此运行时间为O(1)。

空间复杂度为O(1)。
