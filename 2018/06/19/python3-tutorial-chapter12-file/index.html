写入文件

open()函数在打开文件时可以设置打开模式.

mode实参有以下几个可选的值：

  • ‘w’：写入模式
  • ‘r’：读取模式
  • ‘a’：附加模式
  • ‘r+’：读取和写入模式
  • ‘ｗb’：二进制写入模式
  • ‘rb’：二进制读取模式

如果省略了模式实参，也就是之前我们采用的open(filename)这种方式，python将以默认的只读模式打开文件．

以’w’模式打开文件，如果要写入的文件不存在，open()将自动创建它．另外，如果指定的文件已经存在，python将在返回文件对象前清空该文件．
