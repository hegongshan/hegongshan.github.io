Maven常用命令总结

发表于 2021-09-20 | 更新于 2021-10-11 | 分类于 java | 浏览 次
字数统计: 855

对于Maven项目，其目录结构通常为：

配置Java编译器

通过source指定源码版本，而target则指定编译版本。

mvn compile

编译项目源码，生成的*.class文件位于target/classes中。

mvn test

执行测试代码，生成的.class位于target/test-classes中。

mvn test-compile

编译测试代码，但不执行

mvn package

打包结果位于${basedir}/target目录下

打包时，会自动运行测试代码。如果不想运行测试代码，可以添加如下的选项：

mvn install

将jar包安装到本地仓库中，以便在本地其他项目中使用该项目${user.home}/.m2/repository

mvn clean

删除target目录
