Java中的内部类继承

发表于 2019-01-13 | 更新于 2019-01-25 | 分类于 java | | 浏览 次
字数统计: 1.1k

由于Java中的匿名内部类不能被继承，而局部内部类和静态内部类的继承，与普通类继承没有区别。故本文主要描述成员内部类的继承问题。

成员内部类的继承可分为以下三种情况：

  • 内部类及其子类在同一个外围类中
  • 内部类及其子类在不同的外围类中
  • 内部类的子类不是内部类

下面对这三种情况逐一分析

内部类及其子类在同一个外围类中
