Git常用命令总结

现代软件开发离不开团队协作，常用的Git命令必须熟练使用。

clone

将远程仓库克隆到本地。

1.克隆master

2.克隆指定分支

config

  • --global：设置全局变量

示例1：设置全局用户名和邮箱

示例2：设置提交编辑器

  • --list：列出所有的配置项

init

创建一个空的Git仓库，或者初始化一个已经存在的仓库。

当前改变

  1. add：将文件添加到索引中。
  1. mv：等同于Linux中的mv命令，用于文件重命名或者移动文件。
  2. reset：版本回退。
  3. rm：等同于Linux中的rm命令，用于删除文件。

log

查看提交日志。

show

status

查看工作目录的当前状态

branch

1.查看仓库的分支。示例：git branch

  • -D：强制删除分支。示例，删除本地的test分支
  • -m：重命名分支。示例，将本地的test分支重命名为feature分支
  • --list：列出所有的分支

checkout

切换分支。

checkout -b：创建一个新分支，并切换到该分支

commit

  • -m，–message

git diff

git merge

git tag

  • -d，–delete：删除标签
  • -l，–list：列出所有的标签

stash

git fetch
