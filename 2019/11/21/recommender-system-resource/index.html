推荐系统 入坑指南

发表于 2019-11-21 | 更新于 2021-03-05 | 分类于 recommender systems | 浏览 次
字数统计: 1.4k

推荐系统（Recommender Systems，RS）是我硕士期间的研究方向，一路走来，主要靠自己不断摸索。

本文总结了一些和推荐系统相关的资源，比如相关的会议期刊、书籍等，希望能够帮助想要从事RS研究的你，少走一些弯路！
